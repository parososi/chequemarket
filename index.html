Perguntas frequentes

Os pigmentos são confiáveis?

Sim, as siglas seguem a nomenclatura do Colour Index (ex.: PB29 - Ultramarine Blue).

Posso baixar o catálogo?

Este é um protótipo local. Basta salvar o HTML ou publicar em um host estático.

Há suporte a novas marcas?

Sim, basta inserir novos objetos no array paints em script.js com os campos existentes.
